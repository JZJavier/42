# ft_printf

For the ft_printf project of the 42 school cursus, we must recreate the famous C library printf function. This project teaches us about variadic arguments.

- Supported conversions: %, c, s, p, i, d, u, x, X

  
## Format Specifiers

    <thead>
        <tr>
